## Regras Para Registro de Novas Versões

Toda nova versão do GCCI™ deve incluir:

1. Arquivos PDF atualizados
2. Hash SHA-256 e SHA-512
3. Registro neste documento sob nova seção
4. Atualização correspondente no LinkedIn e GitHub Pages

Qualquer versão que não esteja neste registro não é reconhecida oficialmente.

## Declaração de Autenticidade

Todos os hashes aqui registrados constituem referência oficial de auditoria pública.
Qualquer divergência entre arquivos e seus hashes caracteriza alteração, cópia não autorizada
ou tentativa de violação metodológica.
